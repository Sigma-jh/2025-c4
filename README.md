# C4项目改进流程
## V1版本
增加了状态维度，使得状态的输入从184维升到196维
| 类别     | 特征                            | 维度      |
| ------ | ----------------------------- | ------- |
| 雷达     | `scan_range`                  | 180     |
| 目标重复   | `[heading, distance] * 5`     | 10      |
| 目标矢量   | `[dx, dy]`                    | 2       |
| 历史差分   | `[delta_heading, delta_dist]` | 2       |
| 障碍信息   | `[obs_min_range, angle]`      | 2       |
| **总计** |                               | **196** |
